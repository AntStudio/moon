对话框

Moon对话框组件，采用bootstrap的modal对话框为基础，给开发者提供了一套全部基于js的接口，而无需编写对话框的Html代码。

基础对话框


$("#content").dialog({
  title:"测试对话框"
});
注意：这里的#content是对应需要显示内容的选择器

对话框按钮

Moon Dialog提供了对话框按钮配置，可以使用buttons进行配置。

buttons:[
		{
			text:"按钮需要显示的文字",
			class:"按钮的样式",
			click:"点击时需要调用的方法，方法回调的上下文是当前dialog"
		}
]
示例

$("#content").dialog({
	title:"测试对话框",
	buttons:[
		{
			text:"保存",
			class:"btn btn-primary",
			click:function(opts){
				alert("保存");
				$(this).modal("hide");
			}
		},
		{
			text:"取消",
			class:"btn btn-danger",
			click:function(opts){
				$(this).modal("hide");
			}
		}
	]
});
这里在回调方法中可以直接使用上下文this进行对话框的操作，如这里可以直接调用对话框的modal("hide")进行对话框的关闭。 同时还将进行渲染dialog的配置进行传回。
